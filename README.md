# IPHROS Applied Energy 2026
The files in this repository are supplemental material for the paper submitted to Applied Energy entitled "Co-Optimizing the Design and Operation of an Integrated Pumped Hydro Reverse Osmosis System via a GA-MILP Framework: A Case Study for Andalusia, Spain". An offical README will be created at a later date. Until then, the repository can be understood as follows:

## src --> GA-MILP
This folder contains the code making up the GA-MILP discussed in the paper. Running GA_MILP_combined_main.py would start the full GA-MILP optimization.

## src --> Analysis
This folder contains all the python files that, when run, re-create the python-generated figures in the paper. The naming convention of these files correspond to the figures they generate.

## data
This folder contains all of the data used by the "Analysis" python scripts.

As long as the entire repository is downloaded/cloned/etc., any of the python scripts mentioned above should run. Any questions in the interim can be directed to Matthew Haefner, mwh85@cornell.edu
